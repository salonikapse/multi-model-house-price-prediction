# 🏠 Multi-Model House Price Prediction System

This project is an end-to-end Machine Learning system designed to predict house prices using multiple regression algorithms and compare their performance.

🚀 Features:
- Implementation of 10+ regression models (Linear, Ridge, Lasso, Random Forest, XGBoost, LightGBM, ANN, etc.)
- Model evaluation using MAE, MSE, and R² Score
- Automated training and saving of models using pickle
- Flask-based web application for real-time predictions
- Dynamic model selection by user
- Performance comparison dashboard

🛠️ Tech Stack:
- Python
- Scikit-learn, XGBoost, LightGBM
- Pandas, NumPy
- Flask (Web Deployment)
- HTML/CSS (Frontend)

📊 Model Highlights:
Includes advanced models like:
- Random Forest Regressor
- XGBoost Regressor
- LightGBM Regressor
- Artificial Neural Network (MLP)

💡 Use Case:
Helps users estimate property prices and analyze which ML model performs best for housing prediction tasks.

▶️ How to Run:
1. Clone the repository
2. Install dependencies
3. Run training script to generate models
4. Start Flask app: python app.py
5. Open in browser: http://127.0.0.1:5000/
